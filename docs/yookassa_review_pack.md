# YooKassa Review Pack — Hubicx Bot

## 1. Описание бота

**Название:** Hubicx Ai — Nano Banana pro, GPT Image 2, Seedance 2.0

**Username:** @Hubicx_bot

**Платформа:** Telegram Mini App

**Описание:**  
Hubicx — это AI-агрегатор для генерации изображений, видео и текста.  
Пользователь открывает Mini App через Telegram, выбирает модель, вводит описание (промпт) и получает результат генерации.  
Все операции — внутри WebApp, бот используется только для точки входа и стартового сообщения.

**Ссылки:**
- WebApp: https://app.hubicx.ru/app/index.html
- API: https://api.hubicx.ru
- Документы внутри WebApp: Профиль → Документы

## 3. Как работают токены

### Списание
- Токены списываются при создании генерации.
- Стоимость показывается пользователю до подтверждения.
- Если токенов недостаточно — генерация не запускается.

### Возврат
- Токены возвращаются при технической ошибке провайдера.
- Успешные генерации не возвращаются.
- Ошибочное списание можно обжаловать через поддержку.
- Купленные токены не подлежат возврату, кроме случаев технической ошибки при оплате.

### Срок действия
- Токены не сгорают и не имеют срока действия.

## 4. Правила возврата (User-facing)

Текст внутри WebApp (Профиль → Документы → Правила возврата токенов):

> Токены возвращаются на баланс только в случае технической ошибки: генерация не выполнена, но токены списаны.
> Возврат происходит автоматически, если генерация завершилась ошибкой провайдера.
> Токены, потраченные на успешные генерации, не возвращаются, так как результат предоставлен.
> Ошибочное списание можно обжаловать через поддержку.
> Купленные токены возврату не подлежат, за исключением случаев технической ошибки при оплате.
